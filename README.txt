SOBRE O PROJETO:

Projeto que realiza operações de webscrapping, 
transformação de dados, 
criação e consulta de tabelas em bancos de dados, 
e uma aplicação web para consulta de dados via uma API.

TECNOLOGIAS USADAS:

Python: 
- Webscrapping para baixar arquivos na internet
- Transformação de dados dos arquivos baixados
- Criação de um servidor para o backend da aplicação web, 
gerenciando uma API para exportar os dados de uma tabela 
e fazer pesquisas.
-Módulos usados: BeautifulSoup, OS, Pandas, Pdfplumber, 
Zipfile, Requests, http.server.

MYSQL:
- Criação de tabelas
- Importar arquivos CSV para as tabelas criadas
- Consultas nas tabelas para extrair informações

JavaScript, HTML, CSS (Bootstrap), Vue.js:
- Aplicação web recebendo informação da API do servidor
- Exibição de informações dinâmicas em tabela
- Pesquisar informação relevantes pela API

Postman:
- Criação de coleção para testar a funcionalidade da API
- Exportação da coleção em arquvio json
